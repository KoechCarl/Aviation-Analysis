# Aviation Accident Risk Analysis

## Repository Overview
This repository contains all materials for the Aviation Analysis project. The project evaluates historical aviation accident data to identify aircraft characteristics associated with lower operational risk and supports data-driven aircraft acquisition decisions.

---

## Repository Contents

### Data
The `data/` directory contains the aviation accident dataset used for analysis.

- `Aviation_Data.csv` – Aviation accident data used in the notebook.
- `Cleaned_Aviation.csv` - A cleaned version of `Aviation_Data.csv` that was used in the Tableau Dashboards.

---

### Notebooks
The `notebooks/` directory contains the Jupyter Notebook used for analysis.

- `Aviation_analysis.ipynb` – Data cleaning, exploratory analysis, visualizations, and business recommendations.

---

### Dashboards
Interactive visualizations were created using Tableau and published online.

- Tableau Public Dashboard:  
  https://public.tableau.com/app/profile/carl.collins/viz/Aviator-Analysis-Visuals/Dashboard1

## How to Use This Repository
1. Review the notebook in the `notebooks/` folder to understand the analysis and methodology.
2. Explore the interactive Tableau dashboards using the link above for hands-on visualization.
3. View the presentation PDF for a high-level summary of insights and recommendations.
